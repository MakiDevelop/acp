Key Design Decisions
✓ Zero intrusion — acpctl run wraps agents, agents don't know ACP exists
✓ Metadata only — never captures prompt content, only who/when/what-tool
✓ Append-only audit — hash-chain integrity, tamper-detectable
✓ Vendor agnostic — any AI CLI, any model, any framework
✓ OSS first — Apache 2.0, self-hosted, no SaaS dependency
✓ OTel compatible — ACP owns governance model, OTel for trace transport
MVP-1 Scope (Month 1) — What ships first
acpctl register — Register an agent (idempotent by slug+fingerprint)
acpctl list — List all agents with owner, status, last seen
acpctl run -- <cmd> — Wrap any CLI agent with auto register+heartbeat+report
acpctl audit — View append-only audit event log
docker compose up — One command to run API + Postgres
